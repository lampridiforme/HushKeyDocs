Home

Last updated: June 4th, 2018
  • Description: Home page. Contains the common wrapper that all pages should share (navbar and header).
  • Note:
    • File contains sidebar UI only, the top bar is used as a component but to actually change it you need to go into HomeHeader.
  • Dependencies:
    • Components:
      • HomeHeader: Top bar of the page, contains the menu icon and site logo/name
      • MaterialUI:
        • Drawer
          • Mobile sidebar is a temporary drawer, while desktop sidebar is a permanent drawer
        • MenuList
          • Sidebar is a MenuList of elements
        • MenuItem
          • Each element of the sidebar is a MenuItem
        • ListItemIcon
          • Icon for each element in the sidebar
        • ListItemText
          • Text for each element in the sidebar
        • Divider
          • Used to split the sidebar between two categories
        • Hidden
          • Hides menu types depending on the screen size
