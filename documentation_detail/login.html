Login

Last updated: June 4th, 2018
  • Login page.
  • Note:
    • Authentication happens in the handleInputChange method; if there is input, then login is not prevented
  • Dependencies:
    • Components:
      • none
      • MaterialUI:
        • Text
          • Place to enter text
        • Paper
          • Layout System
        • Buttons
          • Navigation Buttons
        • IconButtons
          • Icons that behave as buttons, used for the other login options (Facebook, Google, Github)
        • Divider
          • Divides the alternative login systems with the main login textboxes
