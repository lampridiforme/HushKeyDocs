Tools & Services

Last updated: June 4th, 2018

Conveyor Belt Tools



Continuous Integration

We chose Travis CI for our CI pipeline for the following reasons:

  • Unlimited build time
  • Free student plan
  • Great 3rd party support
  • Modular
  • Easy to set up
  • Prior experience



JavaScript Framework

We chose React for our JavaScript Framework for the following reasons:

  • Members have prior experience
  • Extensive support
  • Supports components
  • Web app compatible



Hosting & Database

We chose Firebase to host our web app and Cloud Firestore as our database management system for the following reasons:

  • Consistent and fast
  • Gzips css/js files
  • Mobile SDK
  • Easy to use
  • Well documented
  • NoSQL: easy to manage
  • Offline support for PWA



Testing Frameworks & Utilities

We chose Mocha and Nightwatch as our testing frameworks and Enzyme, Expect, and JSDOM for the following reasons:

  • Mocha: large support community
  • Simple setup
  • Flexible configuration
  • Enzyme orginally designed to work with React and Mocha
  • Expect: handles assertions
  • JSDOM: render components



Planning Software

We chose Zenhub for task management and to plan our milestones for the following reasons:

  • Members have prior experience
  • Github interface
  • Issues can be filtered by milestones
  • Simple drag and drop functionality



Code Quality Analysis

We chose Code Climate for the following reasons:

  • A great number of supported languages, technologies and frameworks
  • Well-maintained test coverage feature gem
  • Very stable and has a nice UI



Performance Measurement

We chose Lighthouse for the following reasons:

  • Performance tests not just based on assumed best practices
  • Includes tests for Progressive Web Apps, best practices, and accessibility

Web App Tools



Error Tracking

We chose Trackjs as our error tracking software for the following reasons:

  • Includes telemetry timeline to show events leading up to exception
  • Error trends
  • Analyrics dashboard



General Analytics

We chose Google Analytics for the following reasons:

  • Free to use
  • Monitors social networking activity
  • Ability to track mobile users
  • Simple integration and easy to use interface
  • Google analytics helps rate each page on the website



Usability Analytics

We chose Fullstory for the following reasons:

  • Rich visualizations while providing accurate data
  • Keeps track of all the user iterations over the site and application



Component Library

We chose React-Material-UI for the following reasons:

  • Useful components to use
  • All features are free
  • Has support for many themes and supports overriding of css styles
  • Used on React with many users and community
